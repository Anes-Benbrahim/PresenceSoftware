# PresenceSoftware
A comprehensive system designed to automate the process of managing student attendance in schools. This project saves time and effort by streamlining notifications and record-keeping for absent students.

## Features

- **Automated Notifications:** Sends notifications to parents when a student is absent.
- **Absence Archive Management:** Automatically updates the attendance records based on selected absentees via the mobile app.
- **Time-Saving:** Reduces the manual effort required to manage attendance every day.

## Technologies Used

- Desktop & Mobile App Development (Unity Editor)
- Programming Languages (C#)
- Database Management (Firebase)
